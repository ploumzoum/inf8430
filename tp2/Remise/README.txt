Pour lancer le projet:

1 - Aller dans le dossier remise
2 - Faire rouler le NameService.jar
3 - Prendre en note l'adresse affichée sur la console
4 - Ouvrir les serveurs de calcul (Calculator.jar) avec comme paramètres la malice, la capacité et l'adresse du NameService
5 - Ouvrir le répartiteur (Distributor.jar) avec comme paramètre l'adresse du NameService

Si un serveur de calcul est déconnecté, il faut redémarrer le NameService et tous les serveurs de calcul, puis le répartiteur (l'ordre est nécessaire).
